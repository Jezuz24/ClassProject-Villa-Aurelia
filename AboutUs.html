Villa Aurelia is a modern hotel booking system inspired by nature, gothic windows, botanical gardens, and luxury castle-style architecture.

This project uses HTML, external CSS, Tailwind CSS, responsive design, and JavaScript validation. Users can register, login, book rooms, view amenities, submit reviews, and contact customer service.

The administrator has full access to manage guest accounts and booking statuses, while guest/customer accounts have limited access to their own dashboard and confirmations.
